학과: 빅데이터과

01.

인터넷

은 전 세계를 연결하는 국제 정보 통신망으로, 컴퓨터나 스마트폰 같은 디지털 기기로 연결되어 사람들이 정보를 공유할 수 있는 공간입니다.

02.

하이퍼링크

는 인터넷에서 문서 사이를 쉽게 이동할 수 있는 기능입니다.

03.

W3C

는 HTML 표준을 비롯한 웹 표준안을 제작하거나 제안하는 국제적인 웹 표준화 단체 입니다.

04.

웹 브라우저

는 웹에 공개된 정보를 탐색할 수 있게 하는 프로그램입니다.

05.

플러그인

은 사용자 PC에 프로그램을 추가로 설치해 웹 브라우저 기능을 확장하는 방법입니다.

06.

웹 2.0

은 서로 다른 사용자가 함께 새로운 콘텐츠를 창조할 수 있는 시대를 의미합니다.

07.

WHATWG

는 Web Application 1.0 표준을 작성한 단체로, 현재는 단체 이름을 HTML W/G로 변경했습니다. Web Application 1.0 표준은 새로운 웹 표준으로 채택된 후

HTML5

표준이라는 이름으로 변경되었습니다.

08. 웹에서

요청

하는 쪽을 클라이언트(사용자)라고 하며 응답하는 쪽을

서버

(

제공자

)라고 합니다.

09.

URL

은 웹에서 어떤 대상을 구분하는 주소를 의미합니다.

10. 서버 프로그램은

사용자

와 먼 곳에 있어 백엔드 프로그램이라고 합니다. 클라이언트 프로그램은

사용자

와 가까운 곳에 있어

프런트엔드

프로그램이라고 합니다.

11.

마크업

은 웹 페이지의 서식이나 구조를 표현하는 정보입니다.

12.

CSS

는 HTML 페이지에 스타일을 지정하는 스타일시트를 작성할 때 사용하는 언어입니다.

13.

자바스크립트

는 HTML 페이지에서 사용자 반응 등을 처리하는 스크립트를 작성하는 언어로, 표준 명칭은

ECMAScript

입니다.

14.

일렉트론

은 HTML5 기반의 데스크톱 애플리케이션 개발 엔진입니다.

15.

리액트 네이티브

는 페이스북에서 만든 모바일 애플리케이션 개발 엔진입니다.

16.

크로스 플랫폼

은 한 번 개발해서 여러 장치(플랫폼)에 대응하는 것을 의미합니다.

약자 설명.

3. W3C(World Wide Web Consortium)

: 월드 와이드 웹 컨소시엄의 약자로 간단히 말해 웹 세상의 규칙을 만드는 곳이다. 팀 버너스리가 설립하였고, 모든 사람이 문제없이 웹을 사용할 수 있도록 웹 표준 기술을 만들고 제안하는 국제적인 단체라고 함.

7. WHATWG (Web Hypertext Application Technology Working Group)

: 웹 하이퍼텍스트 애플리케이션 기술 워킹 그룹의 약자이다. W3C가 웹의 전반적인 표준을 다룬다면, WHATWG는 특히 HTML과 관련 기술을 '살아있는 표준(Living Standard)으로 계속해서 발전시키는 데 집중하는 단체라고 함.

9. URI (Uniform Resource Identifier)

: 통합 자원 식별자의 약자이다. 인터넷에 있는 자원(문서, 이미지, 영상 등)을 구분하는 고유한 주소를 의미. 우리가 흔히 쓰는 URL(Uniform Resource Locator)은 이 URI에 포함되는 개념이다. URI (식별자): 리소스의 이름. 신분증처럼 그 대상을 고유하게 식별합니다. URL (위치): 리소스가 어디에 있는지 위치를 알려주는 주소. 즉, 모든 URL은 URI이지만, 모든 URI가 URL은 아니므로, URI가 더 큰 개념입니다.

12. CSS (Cascading Style Sheets)

: 캐스케이딩 스타일 시트의 약자이다. 웹 페이지의 디자인과 스타일을 담당하는 언어이다. HTML이 웹 페이지의 뼈대(구조와 내용)를 만든다면, CSS는 그 뼈대에 옷을 입히고 화장을 하는 역할을 함. 글자 색상, 폰트, 배경 이미지, 요소들의 배치 등 웹 페이지의 시각적인 모든 것을 CSS로 꾸밀 수 있음.

감상.

1~2주차 수업을 수강하면서 웹을 구성하는 핵심적인 개념을 학습할수 있었습니다. 1~2주차 수업을 수강하기 이전엔 일상에서 흔히 사용하는 인터넷, 그리고 웹브라우저가 어떤원리와 구조에서 작동하는지 깊이 있게 생각해보았던 경험은 없었지만, 이번 수업시간 그리고 마무리로 학습자료들을 통해 흩어져 있던 개념조각들이 한데 모여 점차 하나의 그림으로 정리할수 있었던 것 같습니다.

아직 부족한점이 많고 모르는점 또한 많지만, 이번 수업을 통해 웹을 단순히 사용자 입장에서만 생각하고, 사용하였던 불과 며칠 전의 저와 달리 앞으로 점차 이해자의 입장으로 한걸음 나아가게 해준 시간이 되었던 것 같습니다.
